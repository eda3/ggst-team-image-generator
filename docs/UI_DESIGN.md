## 2. サイズ仕様

| プリセット | 幅 | 高さ | 用途 |
|-----------|-----|------|------|
| twitter | 1200px | 675px | Twitter投稿用 |
| discord | 800px | 450px | Discord埋め込み |
| custom | 任意 | 任意 | カスタム |

## 3. 色彩設計

```typescript
const DEFAULT_COLORS = {
  // 背景
  background: {
    primary: '#1a1a2e',      // ダークネイビー
    secondary: '#16213e',    // 濃紺
    accent: '#0f3460',       // アクセント青
  },

  // テキスト
  text: {
    teamName: '#ffffff',     // 白
    playerName: '#e94560',   // GGST風赤
    characterName: '#aaaaaa', // グレー
  },

  // 装飾
  decoration: {
    border: '#e94560',       // 赤ボーダー
    divider: '#333355',      // 区切り線
  },
};
```

## 4. フォント

- **チーム名**: Noto Sans JP Bold, 48px
- **プレイヤー名**: Noto Sans JP Bold, 24px
